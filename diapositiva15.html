Introducción al Procesamiento de Datos

Duración:48 horas (24 teóricas, 24 prácticas)
Tipología:Teórica - práctica

Esta asignatura introduce los conceptos fundamentales sobre el manejo y procesamiento de datos. Se abordan temas como el ciclo de vida del dato, la calidad de la información, y el uso inicial de herramientas básicas como hojas de cálculo.

Herramientas Informáticas para Procesamiento de Datos

Duración:96 horas (48 teóricas, 48 prácticas)
Tipología:Teórica - práctica

Se exploran herramientas de software fundamentales como hojas de cálculo, procesadores de texto y presentaciones. Se orienta al uso eficiente de estas herramientas en el contexto del procesamiento de datos.

Pruebas de Software

Duración:48 horas (24 teóricas, 24 prácticas)
Tipología:Teórica - práctica

Este módulo se enfoca en el desarrollo de habilidades para la planificación, ejecución y documentación de pruebas funcionales y no funcionales en software, identificando errores y asegurando la calidad del producto.

Inglés Técnico para Data Science

Duración:96 horas (64 teóricas, 32 prácticas)
Tipología:Teórica - práctica

El curso fortalece el vocabulario técnico y habilidades comunicativas en inglés, específicamente orientadas al ámbito del análisis y procesamiento de datos, lectura de documentación técnica y elaboración de informes.

Competencias Comunicativas

Duración:96 horas (64 teóricas, 32 prácticas)
Tipología:Teórica

Este módulo busca desarrollar la capacidad para comunicarse de forma clara y efectiva en distintos contextos, haciendo énfasis en la argumentación, redacción y comprensión de textos técnicos.

Proyecto Integrador - Prácticas

Duración:288 horas (48 teóricas, 240 prácticas)
Tipología:Práctica

Este espacio permite la integración de los conocimientos adquiridos a través de un proyecto práctico supervisado. Los estudiantes enfrentan retos reales del entorno laboral para aplicar sus competencias.

Visualización de Datos

Duración:96 horas (48 teóricas, 48 prácticas)
Tipología:Teórica - práctica

En este módulo se desarrollan habilidades para representar datos de manera visual utilizando gráficos y paneles interactivos. Se trabaja con herramientas digitales para facilitar la toma de decisiones basada en datos.

Sistematización de Datos Masivos

Duración:144 horas (72 teóricas, 72 prácticas)
Tipología:Teórica - práctica

Este módulo aborda el procesamiento de grandes volúmenes de datos mediante herramientas automatizadas. Los estudiantes aprenden técnicas para recopilar, organizar y almacenar datos masivos de forma eficiente y segura.

Administración de Bases de Datos Relacionales

Duración:144 horas (72 teóricas, 72 prácticas)
Tipología:Teórica - práctica

Los estudiantes adquieren habilidades para diseñar, implementar y administrar bases de datos relacionales, aplicando lenguajes como SQL y normas de integridad y seguridad en el manejo de información.
